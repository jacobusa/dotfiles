# Keyboard Shortcuts

- Shortcuts for mac only
- These are listed from most general (work on most apps) to more specific (likely only work on that app and below) for that OS. They are cumilative as you move down

## macOS

### General

- `CMD + +` ‚Üí Zoom In
- `CMD + -` ‚Üí Zoom Out
- `CMD + C` ‚Üí Copy
- `cmd + v` ‚Üí Paste
- `CMD + TAB` ‚Üí Application Switcher
- `CTRL + P` ‚Üí Move up a selection in a menu
- `CTRL + N` ‚Üí Move down a selection in a menu
- `CTRL + TAB` ‚Üí Go to next tab
- `CTRL + SHIFT + TAB` ‚Üí Go to previous tab
- `CMD + ,` ‚Üí Pull up settings for most apps
- `CTRL + LEFT/RIGHT_ARROW` ‚Üí Switch between active windows
- `CMD + \` ` ‚Üí Next Instance of App
- `CMD + W ` ‚Üí Delete Window
- `CMD + Q ` ‚Üí Quit App
- `CMD + N ` ‚Üí New command (context specific but universal)
- `CMD + OPTION + D ` ‚Üí Toggle home dock visibility
- `CMD + K ` ‚Üí # Global search in some contexts
